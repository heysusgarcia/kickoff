Kickoff
=======

[Kickoff][www.kickoff.pw] Link

- A kickstarter clone dedicated to all things shoes.
- Built using Ruby on Rails and Backbone.js
- Includes User Authentication, ability to upload profile and project images via Paperclip. Used Rails and AWS to store images.
